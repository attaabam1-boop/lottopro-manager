# LottoPro Manager

A professional offline Lotto Management System for Android and Windows built with Flutter, Dart, SQLite, WhatsApp links, and Excel export.

## Android APK

This project now includes an Android runner under `android/`.

Build a debug APK:

```powershell
flutter pub get
flutter build apk --debug
```

The APK will be created at:

```text
build/app/outputs/flutter-apk/app-debug.apk
```

Build a release APK:

```powershell
flutter pub get
flutter build apk --release
```

The release APK will be created at:

```text
build/app/outputs/flutter-apk/app-release.apk
```

If Flutter says platform files need regeneration, run:

```powershell
flutter create --platforms=android .
```
